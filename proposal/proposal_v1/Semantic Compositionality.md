## 1. **Intuitive Definition & Meaning**

Semantic compositionality is the idea that:

* **Complex meanings** arise from **simpler components**
* And the **rules for combining them**

**Meaning = Meaning of parts + How they are combined**

This applies across domains:

* **Language:** “red car” = red + car
* **Vision:** dog + couch + “on” → dog on couch
* **Latent spaces:** attribute factors combine to form new concepts
* **Generative modeling:** combine learned attributes/score fields to create novel, valid outputs

**Core intuition:** If you understand the pieces, you can construct the whole meaning.

---

## 2. **Why It Matters in Generative Modeling**

Semantic compositionality is crucial because generative models often:

* Memorize correlations instead of abstract concepts
* Fail on unseen combinations
* Entangle unrelated attributes
* Confuse local vs global factors

Models with compositionality can:

* Combine attributes reliably
* Generate unseen but meaningful combinations
* Build factors of variation that add predictably
* Support scientific hypotheses where components must be composable (e.g., disease A + disease B)

---

## 3. **Types of Semantic Compositionality**

**a. Attribute Compositionality**
Combine independent attributes (red + small + metallic)

**b. Object Compositionality**
Build scenes from multiple objects (car + road)

**c. Relational Compositionality**
Combine relations with objects (dog *under* table)

**d. Latent Factor Compositionality**
Combine disentangled latent factors (shape + texture + lighting)

**e. Generative Process Compositionality**
Combine generative mechanisms (e.g., score fields, energy terms, SDE components)

---

## 4. **In Generative Image Composition**

A model has semantic compositionality if it can:

* Combine multiple attributes or disease patterns correctly
* Merge local changes without disturbing global structure
* Produce coherent images from independent generative factors
* Synthesize unseen combinations (e.g., TB + silicosis) while preserving anatomy

**Generative composition ≈ additive/composable transformations in latent or score space.**

---

## 5. **Non-Examples — When Compositionality Fails**

Compositionality breaks when:

* Attributes are entangled (glasses → male)
* Local edits distort global structures
* Adding one concept deletes or corrupts another
* The model can't represent independent factors
* Generations rely on memorized correlations
* Score fields interfere destructively (e.g., TB lesions removing rib anatomy)

These failures indicate missing factorization or poor structural understanding.
